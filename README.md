# Web_Store_App_Project

## Описание

Проект Web_Store_App_Project представляет собой простое веб-приложение на Python для отображения страницы контактов и сбора данных через POST-запросы.

С помощью Bootstrap собраны прототипы страниц:
 - Главная (home.html)
 - Каталог (catalogue.html)
 - Категория 1 (category.html)
 - Контакты (contacts.html)

Соответствующие HTML-документы расположены в папке html_files в корне проекта.

Для реализации веб-приложения создан класс MyServer в модуле web_app.py, поддерживающий следующие методы:
- do_GET (Метод для обработки входящих GET-запросов)
- do_POST (Метод для обработки входящих POST-запросов)

Модуль main.py - точка входа.

## Структура


- `main.py` — точка входа, запуск сервера.
- `src/web_app.py` — логика обработки GET и POST запросов.
- `html_files/` — верстка страницы.
- `tests/` — тестирование. 

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/AlexanderKokorinV/Web_Store_App_Project.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Запуск программы

```
python main.py
```

## Тестирование

Проект использует `pytest` для написания и выполнения тестов.
Процент покрытия тестами (coverage rate) расположен в файле index.html

Чтобы запустить тесты, следуйте инструкциям ниже:
1. Убедитесь, что `pytest` установлен. Если нет, установите его командой:
```
pip install pytest
```
2. В корневой директории проекта выполните команду для запуска всех тестов:
```
pytest
```
3. Если необходимо, вы можете указать конкретный файл теста:
```
pytest tests/test_web_app.py
```
Это позволит убедиться, что проект работает корректно и все функции проходят проверку.

4. Для проверки покрытия тестами воспользуйтесь командой:
```
pytest --cov=src --cov-report=html
```
